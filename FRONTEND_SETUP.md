# Frontend Setup Instructions

## Step 1: Install Dependencies

Run this command in your frontend directory:

```bash
cd /Users/uday/Documents/UdayPersonalProjects/expense-tracker/expense-tracker
pnpm add @tanstack/react-query @tanstack/react-query-devtools axios zod react-router-dom react-hook-form @hookform/resolvers date-fns
```

## What Each Package Does:

- **@tanstack/react-query** - Manages server state, caching, and API calls
- **@tanstack/react-query-devtools** - Developer tools for debugging queries
- **axios** - HTTP client for making API requests
- **zod** - Schema validation library
- **react-router-dom** - Client-side routing
- **react-hook-form** - Form state management
- **@hookform/resolvers** - Integrates Zod with React Hook Form
- **date-fns** - Date formatting and manipulation
